CodeAlpha Object Detection

A real-time Object Detection and Tracking system developed using YOLOv8 and ByteTrack. The application uses a webcam to detect and track objects continuously with unique tracking IDs.

Features

- Real-time object detection
- Real-time object tracking
- Webcam-based detection
- YOLOv8 model integration
- ByteTrack tracking algorithm
- Fast and efficient performance

Technologies Used

- Python
- YOLOv8 (Ultralytics)
- ByteTrack
- Computer Vision

How It Works

1. The webcam captures live video.
2. YOLOv8 detects objects in each frame.
3. ByteTrack assigns tracking IDs to detected objects.
4. Objects are tracked continuously across frames.
5. Results are displayed in real time.
