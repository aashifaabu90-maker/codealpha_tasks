CodeAlpha FAQ Chatbot

A web-based FAQ Chatbot developed using Python, Flask, NLTK, and Machine Learning techniques. The chatbot answers frequently asked questions by finding the most relevant response using TF-IDF vectorization and Cosine Similarity.

Features

- Interactive chatbot interface
- Answers frequently asked questions
- Uses Natural Language Processing (NLP)
- TF-IDF based question matching
- Cosine Similarity for response prediction
- Chat history tracking
- User-friendly web interface
- Handles unknown questions gracefully

Technologies Used

- Python
- Flask
- NLTK
- Scikit-learn
- HTML
- CSS

How It Works

1. User enters a question.
2. The chatbot preprocesses the text using NLP techniques.
3. TF-IDF Vectorizer converts text into numerical vectors.
4. Cosine Similarity finds the closest matching FAQ.
5. The chatbot displays the most relevant answer.
6. Previous conversations are stored in chat history.
